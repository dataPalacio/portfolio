Análise de dados com linguagem Python

Este curso foi pensado para unir o melhor dos dois mundos: a base teórica essencial e muita mão na massa! Durante as aulas, pude praticar em laboratórios e projetos que simulam desafios reais que um analista enfrenta no cotidiano profissional – desde a limpeza de dados até a criação de dashboards e relatórios estratégicos.

O mais valioso, para mim, foi como o curso me preparou para encarar situações reais do mercado. Além de trabalhar com ferramentas como SQL, Python e Power BI, aprendi a pensar criticamente, identificar padrões e comunicar resultados de forma clara. Sinto que hoje consigo não apenas analisar dados, mas transformá-los em insights acionáveis para a tomada de decisões.

Laboratórios Práticos:

  • Web Scraping: Extração de dados da web.
  • Count/Frequency Encoding: Técnicas de codificação para variáveis categóricas.
  • Label Encoding: Codificação de variáveis de categorias distintas.
  • One-Hot Encoding: Conversão de categorias em formato binário.

Projetos:

  • Banco Relacional, Python e SQL Para Análise de Dados: Aplicação de SQL e Python para trabalhar com dados relacionais.
  • Análise de Dados de RH: Análise de dados de Recursos Humanos para decisões empresariais.
  • Análise e Limpeza de Dados de Telecomunicações: Tratamento e exploração de dados de telecomunicações.
  • Análise de Dados Para Campanhas de Marketing de Instituições Financeiras: Avaliação de dados financeiros em campanhas de marketing.
  • Análise de Dados Para Estratégias de Crescimento e Retenção de Clientes: Definição de estratégias de retenção com base em dados de clientes.
  • Market Basket Analysis: Análise de padrões de compras e comportamento de clientes.
  • Web Data Analytics: Análise de dados da web para entender o comportamento do usuário.
  • Engenharia de Atributos em Dados Médicos: Criação de atributos relevantes a partir de dados médicos.
  • Pré-Processamento de Dados de Texto Extraídos de Data Lake: Limpeza e preparação de dados textuais extraídos de grandes volumes de dados.
  • Design e Construção de 50 Visualizações: Construção de visualizações eficientes para análise de dados.

Análise de dados com Power BI e clínica de BI

Durante o curso, mergulhei de cabeça no universo do Power BI, explorando tudo desde o planejamento de projetos (como definir objetivos e organizar etapas) até técnicas avançadas de modelagem de dados. Aprendi não só a criar dashboards visualmente impactantes, mas também a transformar dados brutos em insights que fazem a diferença na prática.

O que mais me motivou foi a abordagem do curso: partir do básico para o avançado, sempre com foco em resolver problemas reais. Por exemplo, aprendi a identificar inconsistências em grandes volumes de dados, criar métricas personalizadas e até automatizar processos para ganhar eficiência. Além disso, descobri como apresentar análises de forma clara para diferentes públicos, seja em relatórios técnicos ou em reuniões estratégicas.

Principais tópicos abordados:

  • Planejamento de Projetos de Análise de Dados: definição de projetos, nomenclaturas essenciais, entregáveis e uso do Power BI.
  • Modelagem de Dados: conceitos, modelos transacionais e dimensionais, normalização e análise de erros.
  • Análise de Dados de Data Warehouses: criação de modelos lógicos, uso de ETL e análise de dados.
  • Análise de Dados com DAX: criação de medidas, colunas calculadas, funções DAX e análise temporal.
  • Modelagem Avançada: Slowly Changing Dimensions, Junk Dimensions, Fatos Aditivos e técnicas avançadas.
  • Análise de Dados de Data Lakes: conceitos, diferenças para Data Warehouses e exemplos práticos.
  • Power BI Dataflows e Preparação de Dados Self-Service: criação e automação de Dataflows.
  • Relatórios, Visualizações e Análises: escolha de gráficos, cores e encodings visuais para dashboards.

SQL para data science

Neste curso, explorei o SQL de um jeito que vai muito além do básico: entendi como e por que cada comando funciona, aplicando tudo em projetos que replicam problemas reais de análise de dados. Por exemplo, pratiquei desde consultas simples para extrair métricas até desafios complexos, como ajustar queries lentas e gerenciar grandes bancos de dados.

O que mais me marcou foi descobrir como o SQL é uma ferramenta estratégica – não só para buscar dados, mas para entender histórias por trás deles. Aprendi a identificar inconsistências em tabelas, propor melhorias em modelos existentes e até explicar resultados técnicos para colegas não especialistas, traduzindo números em ações práticas.

Hoje, consigo enxergar o SQL como um aliado para resolver problemas do dia a dia: desde otimizar tempo em tarefas repetitivas até apoiar decisões com bases sólidas de análise.

Principais tópicos abordados:

  • Consultas e Filtragem: consultas simples e avançadas, ordenação, filtros e operadores.
  • Manipulação de Variáveis: categorização, codificação e binarização.
  • Junção de Tabelas: técnicas para combinar dados.
  • Agregação de Dados: funções de agregação para sumarizar e analisar.
  • Funções Avançadas: Window Functions, subqueries e tratamento de datas.
  • Limpeza e Processamento de Dados: tratamento de valores ausentes, outliers e dados defeituosos.
  • Análise de Dados: análises exploratórias e resolução de problemas reais.
  • Otimização de Consultas: melhoria da performance de queries complexas.
  • Criação de Dashboards: Marketing Analytics Dashboard com Google Data Studio.
  • Desafios Práticos: análise de grandes conjuntos, pipelines e dados temporais.
  • Machine Learning com SQL: introdução ao uso de SQL para machine learning.

Este curso forneceu habilidades práticas e teóricas essenciais para o trabalho com grandes volumes de dados, análise exploratória e otimização de processos, essenciais para quem trabalha na área de dados.
